# Claude Code Configuration

This directory contains configuration settings for Claude Code.

## Directory Structure

- **`agents/`** - Custom agent configurations and prompts
- **`commands/`** - Custom command definitions
- **`settings.json`** - Main Claude Code settings file

## Settings File

The `settings.json` file contains your Claude Code preferences and configuration options. This includes tool settings, model preferences, and other customizations.

## Documentation

For complete documentation on Claude Code settings and configuration options, visit:
